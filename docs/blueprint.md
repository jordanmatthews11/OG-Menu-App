# **App Name**: Storesight Companion

## Core Features:

- Dynamic Category Catalog: Display retail categories with department, name, number, description, and example brands, filtered by country.
- Store List Catalog: Show standard store lists with name, retailer, weekly, and monthly quota.
- Order Form Submission: Select categories, store lists, and booster retailers to submit an order form. Writes data to Google Sheet.
- Support Request Form: Rebuild the internal support request form with dynamic logic for collections and support requests.
- Form Routing: Route form requests to different staff based on a reasoning tool that takes request type and urgency into account.
- International Support Toggle: Enable country-specific views and submissions using a country toggle.
- Data Export: Export form responses to a Google Sheet, retaining logic branching.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke a sense of professionalism and reliability. Purple is often associated with business, but can also reference scientific applications and knowledge work.
- Background color: Light purple (#F3E5F5), a very desaturated version of the primary color to offer a gentle backdrop that doesn't distract.
- Accent color: Blue (#3F51B5), an analogous color to purple but offering a strong contrast in both saturation and brightness for focus on CTAs.
- Font: 'Inter' (sans-serif) for both headlines and body text, because it offers a modern, machined, objective, neutral appearance and is suitable for both headlines and body text, without need for a pairing. Note: currently only Google Fonts are supported.
- Simple, geometric icons for categories and actions, maintaining a clean and professional aesthetic.
- Consistent, grid-based layout with clear separation of sections for easy navigation and readability.
- Subtle transitions and animations for interactive elements to provide feedback without being distracting.
